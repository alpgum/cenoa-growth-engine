## ⚠️ Critical Guidelines

- **ALL copy must be Egyptian Arabic dialect (عامية مصرية)** — NOT Modern Standard Arabic (فصحى)
- **NEVER use:** crypto, cryptocurrency, تداول, عملات رقمية, بورصة — frame as "digital wallet" / "dollar account"
- **RTL layout** throughout — all text right-aligned, UI mirrored
- **Mobile-first** design (Egypt is >95% mobile traffic)
- **Page load target:** <3 seconds (Egyptian mobile networks can be slow)

## 10) Design Notes

### Layout
- **Direction:** RTL (right-to-left) — all CSS `direction: rtl; text-align: right;`
- **Mobile-first:** Design at 375px width, scale up to tablet/desktop
- **Sticky mobile CTA:** Bottom bar with CTA button, always visible on scroll
- **Section order:** Hero → Benefits → Trust Signals → Social Proof → FAQ → Footer
- **Deep link:** CTA buttons should deep-link to app with `country=EG` pre-filled

### Typography
- **Primary font:** [Cairo](https://fonts.google.com/specimen/Cairo) — clean, modern Arabic/Latin font with multiple weights
- **Secondary font:** [Tajawal](https://fonts.google.com/specimen/Tajawal) — lighter alternative for body text
- **Heading sizes:** H1 32px mobile / 48px desktop, H2 24px / 36px, Body 16px / 18px
- **Line height:** 1.6–1.8 for Arabic (Arabic script needs more vertical space)
- **Font weights:** Bold (700) for headlines, Medium (500) for subheads, Regular (400) for body

### Color Scheme
- **Primary:** Cenoa brand blue (use existing brand color — high contrast for CTA)
- **CTA button:** High-contrast brand accent (e.g., bright blue/green on white)
- **CTA text:** White on brand color
- **Background:** White (#FFFFFF) primary, Light grey (#F8F9FA) for alternating sections
- **Text:** Near-black (#1A1A2E) for body, darker shade for headlines
- **Trust section:** Light blue or light grey background to set apart
- **Accents:** Green for savings/positive messaging, brand blue for interactive elements

### Imagery
- Use culturally relevant imagery — Egyptian freelancers, Cairo/Alexandria skyline, local context
- **Do NOT** just translate Turkish creative assets — create Egypt-specific visuals
- App screenshots should show Arabic UI (if available) or English UI with Arabic overlay
- Avoid stock photos of Western people — use MENA-representative imagery

### Performance
- **Target load time:** <3 seconds on 3G (Egyptian mobile networks)
- Lazy-load images below the fold
- Use WebP with JPEG fallback
- Minimize JavaScript — critical CSS inline
- Compress all assets, limit total page weight to <1MB

### Accessibility
- Minimum contrast ratio 4.5:1 for text
- Touch targets ≥48px for mobile CTA buttons
- Arabic `lang="ar-EG"` attribute on HTML element
- Logical reading order in DOM matches visual RTL layout

### Tracking Implementation
- GA4 events per competitive-positioning.md §7:
  - `view_arabic_lp` (page load)
  - `scroll_50` / `scroll_90`
  - `click_cta_download` (with `cta_variant` parameter)
  - `faq_open_[topic]` (fee / trust / availability / speed / eligibility)
  - `click_trust_partner_expand`
- UTM structure: `utm_source=meta&utm_medium=cpc&utm_campaign=eg_arabic_lp&utm_content=[headline_variant]`
- Amplitude: pass `landing_variant`, `geo=EG`, `language=ar-EG` as properties
