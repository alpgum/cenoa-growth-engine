## Global Specs (All Markets)

| Parameter | Requirement |
|---|---|
| **Video aspect ratio** | 9:16 (vertical, mobile-first) |
| **Video duration** | 15–30 seconds |
| **Video format** | MP4, H.264, ≤30MB |
| **Video resolution** | 1080×1920 minimum |
| **Static dimensions** | 1080×1080 |
| **Carousel card dimensions** | 1080×1080, 3–5 cards per set |
| **Static/carousel format** | PNG, ≤5MB per card |
| **Captions/text overlay** | Required on ALL video (sound-off readable) |
| **Safe zone** | Keep critical text/CTAs within 90% center (Meta ad UI overlaps edges) |
| **Brand elements** | Cenoa logo, brand blue, consistent CTA button style |
| **Delivery** | Shared drive folder per market, named per convention above |
