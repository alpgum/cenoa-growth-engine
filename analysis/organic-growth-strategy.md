## 4. Synergies Between Pillars

```
Content → Social: Blog articles become social posts (carousels, short videos)
Social → Referral: Social proof triggers word-of-mouth sharing
Referral → Content: "I saved $X" cards become UGC content for social + blog testimonials
LP Optimization → All: Higher CTA rate multiplies every traffic source
```

**Concrete example:** "Payoneer Alternatives" blog post (SEO) → cut into TikTok comparison video (Social) → video viewer shares with freelancer friend (Referral) → friend lands on optimized LP (CTA) → installs.

## 7. Measurement Plan — Separating True Organic from Misattributed Paid

| Method | What It Solves |
|---|---|
| **AppsFlyer OneLinks per channel** (social, referral, email) | Attributes installs that currently fall into "(none)" |
| **Deferred deep links** for web→app flows | Captures blog→app store→install journey |
| **Promo codes per channel** (e.g., "TIKTOK50", "BLOG10") | Low-tech attribution backup |
| **"How did you hear about us?" onboarding survey** | Self-reported source for unattributed installs |
| **Paid on/off holdout test** | Pause paid in one geo for 1 week; measure organic delta to find true baseline |
| **GA4 ↔ Amplitude cross-reference** | Match organic session growth % with install growth % weekly |

**Weekly dashboard:** Track organic installs, referral installs, social installs, and unattributed installs as four separate lines. Organic strategy succeeds only if the attributed organic lines grow while unattributed shrinks (better tracking) or stays flat (genuine new volume).
